# TV Show Calendar

Auto-updating iCalendar feed for TV show episodes. Subscribe once, get automatic updates.

## Subscribe

**Calendar URL:** `https://kz-ai-dev.github.io/tv-show-calendar/tv-calendar.ics`

### Google Calendar
1. Open Google Calendar → Click "+" next to "Other calendars"
2. Select "From URL"
3. Paste: `https://kz-ai-dev.github.io/tv-show-calendar/tv-calendar.ics`
4. Click "Add calendar"

### Apple Calendar
1. File → New Calendar Subscription
2. Enter: `https://kz-ai-dev.github.io/tv-show-calendar/tv-calendar.ics`
3. Click OK

## Details

- Updates daily at 6 AM UTC
- Shows episodes in next 90 days
- Data from TVmaze API
- Auto-syncs to your calendar
